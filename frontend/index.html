Workflow

From raw sentence to labeled entities

01

Input and preprocessing

The UI sends text to Flask, which mirrors your Colab inference path: normalize, lowercase, split words, tokenize with `is_split_into_words=True`.

02

Model prediction

The saved model is loaded from the backend folder and returns first-subtoken word-level predictions with BIO labels.

03

Presentation and storage

Frontend renders highlighted entities, a token table, and saves the full run for dashboards, exports, and future user-specific persistence.
